# Travel Landing Page

A modern travel website landing page built using **HTML5** and **CSS3**. This project demonstrates the fundamentals of front-end web development through a clean, responsive interface and a multi-page website structure.

## Features

- Responsive navigation bar
- Multi-page website
- Destination showcase pages
- Clean and modern user interface
- Semantic HTML5 structure
- CSS3 styling

## Technologies

- HTML5
- CSS3

## Project Structure

```
travel-landing-page/
├── index.html
├── style.css
├── README.md
├── LICENSE
├── pages/
└── assets/
```

## Getting Started

1. Clone the repository.
2. Open `index.html` in any modern web browser.
